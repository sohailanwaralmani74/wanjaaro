    <p>Absolutely. The preview table is fully editable. You can change values, add or delete rows, then generate the Excel file from the modified data.</p>

    <p>Yes. Investment transactions are fully supported, including fields like Action (Buy/Sell), Security, Quantity, Price, and Commission.</p>
