The Investment Return Formula

Understanding the return on investment formula helps you measure and compare investment profitability.

Total Return = (Ending Value − Starting Value) / Starting Value × 100 Where: Ending Value = current or exit value, Starting Value = initial investment amount
Annualized Return = (1 + Total Return)1/n − 1 Where: n = number of years

For example, if you invested $10,000 and your investment is now worth $15,000 over 5 years, your total return is 50% and your annualized return is 8.45% — meaning your investment grew at an average annual rate of 8.45% over that period.

Total Return vs Annualized Return — What's the Difference?

MetricDescriptionBest Use Case
Total Return Overall percentage gain or loss over the entire period Measuring total profitability
Annualized Return Average annual growth rate over the period Comparing investments of different durations
Simple Average Return Arithmetic mean of annual returns Quick overview (can be misleading with volatility)

* Annualized return is the most widely used metric for comparing investment performance across different time periods because it accounts for compounding and smooths out volatility.

Frequently Asked Questions About Investment Returns

1. What is the difference between total return and annualized return?
Total return is the overall percentage gain or loss over the entire investment period. Annualized return is the average annual growth rate — it allows you to compare investments of different durations. This investment return calculator shows both.
2. What's a good annualized return?
A "good" annualized return depends on the investment type and your goals. Historically, the US stock market has returned about 7‑10% annually over long periods. Bond markets return 3‑5%, and real estate varies by location. Higher returns usually come with higher risk.
3. Can investment returns be negative?
Yes. If your investment lost value, both your total return and annualized return will be negative. This indicates you lost money on your investment over the period.
4. How do I calculate total return on my investment?
Use the return on investment formula: (Ending Value − Starting Value) / Starting Value × 100. This investment return calculator does the math for you instantly.
5. How does annualized return differ from CAGR?
Annualized return and CAGR (Compound Annual Growth Rate) are the same thing — both measure the average annual growth rate assuming steady compounding. This rate of return calculator uses the same formula as CAGR for annualized returns.
6. Can I use this calculator for stocks and mutual funds?
Yes. This investment performance calculator works for any type of investment — stocks, mutual funds, ETFs, real estate, or your entire portfolio. Just enter the starting and ending values.
7. Why do I need to know my annualized return?
Annualized return allows you to compare investments of different durations on a level playing field. It's the industry standard for measuring and reporting investment performance over time.
